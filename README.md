# BrumaCafe Analytics – Event Profitability & Pricing Strategy

## 📌 Business Problem
Event-based businesses often struggle to understand real profitability due to variable costs, inconsistent pricing, and lack of data visibility.

## 📊 Objective
Analyze event data to improve pricing strategy, reduce costs, and increase profitability.

## 🧾 Dataset
The dataset includes:
- Number of clients per event
- Drinks sold
- Cost of supplies (coffee, milk, cups)
- Transportation and operational costs
- Event type and location

## 🛠 Tools Used
- Power BI (Dashboard & visualization)
- SQL (Data analysis)
- Python (Data processing)
- Excel (Data preparation)

## 🔄 Process
1. Data cleaning and transformation  
2. Cost and revenue modeling  
3. KPI creation  
4. Dashboard visualization  

## 📈 Key Insights
- High revenue events are not always the most profitable  
- Small events can generate higher margins  
- Transportation and supply costs significantly impact profit  

## 💰 Business Impact
- Improved pricing strategy  
- Better cost control  
- Increased profitability per event
