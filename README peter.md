##  Resumen Ejecutivo: Impacto del Análisis de Datos

Tras procesar los datos de eventos de **Brumacafe.gt** con Python y visualizarlos en Power BI, se derivan las siguientes conclusiones estratégicas para el crecimiento del negocio:

1. **Segmento de Alta Rentabilidad:** Los eventos sociales (Bodas) en **Antigua Guatemala** representan el **45% de los ingresos totales**, con un margen de utilidad superior debido a la baja sensibilidad al precio en café de especialidad.
2. **Optimización de Inventario:** El análisis de consumo indica que el **60% de los invitados** prefiere bebidas frías en eventos de día, lo que permite optimizar la compra de hielo y suministros específicos.
3. **Eficiencia Logística:** Se identificó que los eventos de menos de 50 personas tienen costos fijos proporcionalmente altos; se recomienda implementar un cargo logístico base para mantener la rentabilidad mínima del 55%.
4. **Escalabilidad:** La automatización del pipeline de datos (JSON ➔ Python ➔ Power BI) permite al equipo de Bruma Café monitorear su ROI mensual en menos de 5 minutos.
